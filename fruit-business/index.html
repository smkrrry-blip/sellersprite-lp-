🍎 果物ビジネス管理システム

Telegram → Googleスプレッドシート 自動記録

必要なもの

  • Telegramアカウント（iPhoneにインストール済み）
  • Googleアカウント（Gmailで使用中のもの）
  • 所要時間：約20〜30分（初回のみ）

STEP 1：Telegramボットを作る

  1. TelegramでBotFatherを検索 → @BotFather
  2. /newbot と送信
  3. ボット名を入力（例：Fruit Business Bot）
  4. ユーザー名を入力（例：myfruit_bot）※末尾は bot が必須
  5. 表示される トークン（長い文字列）をメモ帳にコピーして保管

STEP 2：チャンネルを3つ作る

iPhone Telegramで：右上の✏️ → 「新しいチャンネル」

チャンネル名用途
🍎 仕入れ市場での仕入れを記録
💰 売上注文が来たら記録
💸 経費充電代・梱包材などを記録

種類は「プライベート」でOKです。

STEP 3：ボットをチャンネルに追加する

  1. 各チャンネルを開く → チャンネル名をタップ
  2. 「管理者を追加」
  3. 作ったボット名を検索して追加
  4. 3つのチャンネル全部に追加する

STEP 4：Googleスプレッドシートを作る

  1. sheets.google.com にアクセス
  2. 「+」で新しいシートを作成
  3. URLの /d/ のあとの文字列が スプレッドシートID

例：...spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit

STEP 5：Google Apps Scriptにコードを貼る

  1. スプレッドシートを開く
  2. 「拡張機能」→「Apps Script」
  3. 最初のコードを全部削除
  4. 上の「💻 Code.gs」タブのコードをコピーして貼り付け
  5. 「＋」で新しいファイルを追加 → 名前 SetupSheets
  6. 「📊 SetupSheets.gs」タブのコードをコピーして貼り付け

STEP 6：設定を書き換える

Code.gsの一番上の CONFIG を書き換えます：

  • BOT_TOKEN：STEP1で取得したトークン
  • SPREADSHEET_ID：STEP4で確認したID
  • CHANNEL：あとで埋める（STEP9で確認）

STEP 7：シートを自動作成する

  1. Apps Scriptの画面で、上部ドロップダウンから setupAllSheets を選択
  2. 「▶ 実行」をクリック
  3. 権限許可画面 → 「許可」→ Googleアカウントでログイン → 「詳細」→「安全でないページへ移動」→「許可」
  4. スプレッドシートに戻ると5つのシートが作られている

STEP 8：Webhookを登録する

  1. 「デプロイ」→「新しいデプロイ」
  2. 「種類を選択」→「ウェブアプリ」
  3. 「アクセスできるユーザー」→ 全員
  4. 「デプロイ」→ 表示されたURLをコピー
  5. ドロップダウンで setWebhook を選択 → 「▶ 実行」
  6. ログに {"ok":true} が出ればOK

STEP 9：チャンネルIDを確認する

  1. 各チャンネルで何かメッセージを送信する
  2. getUpdates を実行 → ログに表示されるJSONを確認
  3. "chat":{"id": の数字（例：-1001234567890）がチャンネルID
  4. Code.gsの CONFIG.CHANNEL に3つのIDを貼り付ける
  5. 「デプロイ」→「デプロイを管理」→ 鉛筆アイコン → 「新しいバージョン」→「デプロイ」
✅ 動作確認：仕入れチャンネルに「マンゴー 5kg 1500」と送ってみてください。ボットが返信してスプレッドシートに記録されたら完成です！

🍎 仕入れチャンネル

書式：果物名 　重量kg 　金額 　[市場名]
マンゴー 10kg 3000 中央市場
みかん 20kg 2000
ぶどう 5kg 4000 農家直送

→ 果物名・重量・単価(/kg)・仕入れ金額・市場名が記録されます

💰 売上チャンネル

書式：顧客名 　商品×kg 　合計金額 　支払い方法 　支払い状態
田中花子 マンゴー×2.5 3500 PayPay 済
山田太郎 みかん×3 マンゴー×1 2800 現金 未
佐藤美咲 ぶどう×2 4000 振込 済
支払い方法現金 / PayPay / 振込 / 銀行
支払い状態済（受け取り済み）/ 未（未払い）

→ 顧客マスタも自動更新。「未」にすると未収金として管理されます

💸 経費チャンネル

書式：カテゴリ 　金額 　[メモ]
充電 1200
充電 800 コンビニ急速充電
梱包資材 1500 みかん箱20枚
市場交通費 300

📊 ダッシュボードの見方

項目内容
今月の売上今月の売上合計（自動計算）
今月の仕入れ今月の仕入れコスト合計
今月の経費充電・梱包材などの合計
今月の利益売上 − 仕入れ − 経費（マイナスは赤表示）
未収金合計「未」にした注文の合計
未収金リスト未払いの顧客名と金額の一覧

⚠️ トラブルシューティング

問題対処
ボットが返信しないsetWebhookを再実行
シートに記録されないチャンネルIDを再確認
権限エラーApps Scriptの権限を再度許可
数値がおかしい半角数字で入力（全角NG）

if (parts.length < 4) { return '❌ 形式エラー\n例: 田中花子 マンゴー×2.5 みかん×1 3500 PayPay 済'; } const customerName = parts[0]; let amount = 0, paymentMethod = '', paymentStatus = '未'; const products = []; const paymentKeywords = ['現金','paypay','PayPay','PAYPAY','振込','銀行']; const paidKeywords = ['済','完了','ok','OK']; const unpaidKeywords = ['未','未払','未払い']; for (let i = 1; i < parts.length; i++) { const p = parts[i]; if (/^\d[\d,]*$/.test(p)) { amount = parseFloat(p.replace(/,/g, '')); } else if (paymentKeywords.some(k => p.toLowerCase() === k.toLowerCase())) { paymentMethod = p; } else if (paidKeywords.some(k => p.toLowerCase() === k.toLowerCase())) { paymentStatus = '済'; } else if (unpaidKeywords.some(k => p.toLowerCase() === k.toLowerCase())) { paymentStatus = '未'; sheet.appendRow([formatDate(date), customerName, productsStr, amount, paymentMethod, paymentStatus, '未']); updateCustomer(ss, customerName, amount, paymentStatus); return ['✅ 売上記録完了', `顧客: ${customerName}`, productsStr ? `商品: ${productsStr}` : '', `金額: ¥${amount.toLocaleString()}`, `支払: ${paymentMethod || '未設定'} ${paymentStatus}`].filter(Boolean).join('\n'); } // ── 経費記録 ────────────────────────────────────────────────── function handleExpense(ss, text, date) { const parts = text.split(/\s+/); if (parts.length < 2) return '❌ 形式エラー\n例: 充電 1200'; const category = parts[0]; const amount = parseFloat(parts[1].replace(/[,円]/g, '')); const memo = parts.slice(2).join(' '); if (isNaN(amount)) return '❌ 金額を正しく入力してください。'; const sheet = ss.getSheetByName('経費'); sheet.appendRow([formatDate(date), category, amount, memo]); return ['✅ 経費記録完了', `${category}: ¥${amount.toLocaleString()}`, memo ? `メモ: ${memo}` : ''].filter(Boolean).join('\n'); } // ── 顧客マスタ更新 ──────────────────────────────────────────── function updateCustomer(ss, name, amount, paymentStatus) { const sheet = ss.getSheetByName('顧客マスタ'); const data = sheet.getDataRange().getValues(); const unpaid = paymentStatus === '未' ? amount : 0; for (let i = 1; i < data.length; i++) { if (data[i][0] === name) { sheet.getRange(i+1, 3).setValue(data[i][2] + 1); sheet.getRange(i+1, 4).setValue(data[i][3] + amount); sheet.getRange(i+1, 5).setValue(data[i][4] + unpaid); sheet.getRange(i+1, 6).setValue(formatDate(new Date())); return; } } sheet.appendRow([name, '', 1, amount, unpaid, formatDate(new Date())]); } // ── Telegramにメッセージ送信 ────────────────────────────────── function sendMessage(chatId, text) { UrlFetchApp.fetch( `https://api.telegram.org/bot${CONFIG.BOT_TOKEN}/sendMessage`, { method: 'post', contentType: 'application/json', payload: JSON.stringify({ chat_id: chatId, text: text }) } ); } // ── Webhook登録（初回のみ実行） ──────────────────────────────── function setWebhook() { const scriptUrl = ScriptApp.getService().getUrl(); const res = UrlFetchApp.fetch( `https://api.telegram.org/bot${CONFIG.BOT_TOKEN}/setWebhook?url=${encodeURIComponent(scriptUrl)}` ); Logger.log(res.getContentText()); } // ── チャンネルID確認用 ───────────────────────────────────────── function getUpdates() { const res = UrlFetchApp.fetch( `https://api.telegram.org/bot${CONFIG.BOT_TOKEN}/getUpdates` ); Logger.log(res.getContentText()); } // ── 日付フォーマット ────────────────────────────────────────── function formatDate(date) { return Utilities.formatDate(date, 'Asia/Tokyo', 'yyyy/MM/dd'); }
